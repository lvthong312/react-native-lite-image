# react-native-lite-image-view

Support for View Image

## Installation


```sh
npm install react-native-lite-image-view
```


## Usage


```js
import { LiteImageViewView } from "react-native-lite-image-view";

// ...

<LiteImageViewView color="tomato" />
```
